# 🛡️ The CIA Triad: The Foundation of Security
> **Professor Messer Section 1.1**

The CIA Triad is the "North Star" of cybersecurity. Every security tool or policy we implement is designed to protect at least one of these three pillars.

---

## 🤫 1. Confidentiality
**Goal:** Keep private data *private*. Only authorized eyes should see it.

* **Encryption:** The heavy lifter. Scrambling data so it's unreadable without a key. 🔐
* **Access Controls:** Restricting data based on "Need to Know" or job roles.
* **Authentication Factors:** Proving you are who you say you are (MFA/Passwords).

## ✅ 2. Integrity
**Goal:** Ensure data is accurate and hasn't been "messed with" during transit or storage.

* **Hashing:** Think of it as a "digital fingerprint." If the hash changes, the data was tampered with! 🖐️
* **Digital Signatures:** Proves who sent the data and that it hasn't changed.
* **Non-repudiation:** No one can say "It wasn't me!" because we have cryptographical proof they sent it.

## ⚡ 3. Availability
**Goal:** Systems must be UP and running when users need them. 

* **Fault Tolerance:** Adding redundancy (extra power supplies, RAID) so if one thing breaks, the system stays online.
* **Patch Management:** Keeping software updated so bugs don't crash the system or create holes for hackers.
