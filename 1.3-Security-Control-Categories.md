## 🧭 Overview
Every security control is classified by **how it is implemented**. When analyzing a control, ask yourself: *Is this done by technology, policy, people, or physical means?*

## 🛠️ The Four Categories

### 💻 Technical
Controls implemented through technology. These run automatically without human intervention.
* **Firewalls:** Hardware or software blocking traffic.
* **Antivirus software:** Automated scanning and remediation.
* **OS policies:** System-level restrictions (e.g., password complexity).

### 📄 Managerial
The "Rulebook." Policies, procedures, and documentation that govern the security program.
* **Acceptable Use Policies (AUP):** What employees can/can't do.
* **Risk Management:** Documentation identifying business threats.
* **Incident Response Procedures:** The plan for when things go wrong.

### 👥 Operational
Controls executed by **people**. These require ongoing human action to stay effective.
* **Security guard patrols:** Physical oversight by staff.
* **Awareness training:** Teaching employees to spot phishing.
* **Walkthroughs:** Periodic physical checks of the facility.

### 🧱 Physical
Measures that restrict physical access to buildings, rooms, or devices.
* **Locks and fences:** Traditional barriers.
* **Badge readers:** Electronic entry points.
* **Security cameras:** Visual surveillance.

> **Exam Tip:** These categories answer **how** a control works. The types (Preventive, Detective, etc.) answer **what** it does!
