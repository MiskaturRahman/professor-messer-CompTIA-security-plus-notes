# 🎭 Security Control Types
**Course:** CompTIA Security+ | Professor Messer  
**Domain:** 1 — General Security Concepts  
**Topic:** The Six Functional Types of Security Controls

---

## 🧭 Overview
While categories describe *how* a control is built, **types** describe its function during an incident. 



## 🛡️ The Six Control Types

1. **🚫 Preventive:** Stops an event *before* it happens. (e.g., Firewall, Locked door).
2. **⚠️ Deterrent:** Discourages attackers (Psychological). (e.g., "Warning" signs, Cameras).
3. **🔍 Detective:** Records that a breach happened. (e.g., Logs, Motion sensors, SIEM alerts).
4. **🩹 Corrective:** Fixes damage *after* the fact. (e.g., Backups, Fire extinguishers, Patches).
5. **🔀 Compensating:** A temporary "Plan B" substitute. (e.g., A generator during power loss).
6. **📜 Directive:** Guides behavior through instruction. (e.g., Compliance checklists, Policies).

---

## 🥊 Key Distinction: Preventive vs. Deterrent
| Feature | Preventive | Deterrent |
| :--- | :--- | :--- |
| **Does it block?** | **Yes** — Active barrier | **No** — Only discourages |
| **Mechanism** | Physical/Technical | Psychological |
| **Example** | A locked gate | A "Beware of Dog" sign |

---

## 🧪 Combining Categories & Types
Real-world controls usually have both a Category and a Type:

- [x] **Firewall Rule:** Technical + Preventive
- [x] **Security Training:** Operational + Directive
- [x] **Motion Detector:** Physical + Detective
- [x] **Backup Restore:** Technical + Corrective
